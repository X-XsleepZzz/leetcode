1. Two Sum

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

step1
```py
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        result_list = []

        for i in range(len(nums)):
            for j in range(i + 1 , len(nums)):
                if nums[i] + nums[j] == target:
                    result_list.extend([i, j])
                    return result_list

```

numsの要素数をnとする。
時間計算量: for文の二重ループで外側がn回、内側が平均でn/2回のループなのでO(n^2)
空間計算量: result_listに2個要素が入るくらいなので、O(1)
とりあえず、ぱっと思いついた方法だが、nums.lengthの最大値が10^4なので時間計算量がn^2は遅い。たしか、pythonは10^7の計算量で1秒くらいだったはず。
2重ループを避けるためにもう一回考える。

```py
import itertools

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        for (i, val1), (j, val2) in itertools.combinations(enumerate(nums), 2):
            if val1 + val2 == target:
                return [i, j]
                
```

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        num_to_index = {}
        for index, num in enumerate(nums):
            if target - num in num_to_index:
